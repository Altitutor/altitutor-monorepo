# Code-defined UCAT generation gates

UCAT AI generation uses code-defined hard gates for non-negotiable format and answer-validity rules, while admin settings control prompts, allowed models, budgets, and tunable thresholds. This keeps UCAT section rules such as fixed question counts, answer-option shapes, and syllogism structure from being accidentally disabled while still allowing generation quality, cost, and model behavior to be tuned over time.
